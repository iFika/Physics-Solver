# PhysicsSolver
PhysicsSolver adalah package NPM beguna untuk menyelesaikan soal Fisika Dengan 5+ Rumus.
## Contoh
```js
const fsolve = require("physicsSolver")
console.log(fsolve.Kecepatan.KecepatanRataRata(Berapa1, Berapa2)
```
Output
```xaml
(v) = (t)/(s)
v = 2
```
